🏨 StayEase

A clean, responsive front-end landing page for a hotel booking experience built using HTML, CSS, and JavaScript. This project showcases a simple UI to present listings and interactions for StayEase — a hotel/stay booking interface concept.

Currently this repository contains only the static UI files: index.html, style.css and script.js. It’s a frontend prototype (no backend or database).

🚀 Features

✨ Modern, minimalistic UI design
📱 Fully responsive layout compatible with desktop & mobile
⚙️ Interactive JavaScript behavior (if any)
🧠 Simple and beginner-friendly codebase
