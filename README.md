# 🏨 StayEase

> A modern and responsive hotel booking landing page built using **HTML, CSS, and JavaScript**.

StayEase is a clean front-end prototype designed to showcase a stylish hotel/stay booking interface.  
This project focuses on delivering a smooth user experience with a minimal UI and responsive layout.

---

## ✨ Features

- 🎨 Modern and minimalistic UI design
- 📱 Fully responsive for mobile, tablet, and desktop
- ⚡ Smooth interactive JavaScript functionality
- 🧩 Simple and beginner-friendly code structure
- 🚀 Fast and lightweight static frontend project
